#OPENCV를 이용한 얼굴인식 APP
---

#Features
* 촬영
* 얼굴/눈/웃음 인식 
* 사진 학습 
* 얼굴 분류 

#Description
* cam.py

    캠 ON 

* faceDetection.py
    얼굴 인식
    
* faceSmileEyeDetection.py
    얼굴/눈/미소 인식

* faceDataset.py
    얼굴 촬영(촬영된 사진은 dataset폴더에 저장)

* faceTraining.py
    얼굴 학습(trainer/trainer.yml)
   
* faceRecognition.py
    학습된 변수를 이용하여 사람 얼굴 분류
